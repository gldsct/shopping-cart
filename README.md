# ShopCart

ShopCart is a web app created with React (scaffolded with Vite).

The app uses mock data to populate items in the shopping section. The mock data is not obtained through an API but through a locally-stored variable (in clothingItems.js); this was primarily because publicly-available APIs provided simple, generic data and had no provisions for the customizations I sought.

## Purpose

This project helped gain familiarity with creating web apps with React, and also helped practice and reinforce the recently-learned concept of client-side routing and single page applications (SPAs).

## Design Inspiration

The design for the app was inspired from [E-commerce Fashion Website](https://dribbble.com/shots/26730142-E-commerce-Fashion-Website) by [Farzan Faruk (farzanfaruk) on Dribbble](https://dribbble.com/farzanfaruk).
